An Electron-based optimization tool designed for **Arknights: Endfield**. It provides features like **Battery Optimization Calculator** and **Auto-Update**.

## Key Features

- **Battery Optimization Calculator**: Mulung 2-splitter/3-merger algorithm calculator.
- **Auto-Update**: Checks for updates on startup and provides changelogs.

## Getting Started

### Prerequisites

- Node.js (v24 or higher)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/NERDHEAD-lab/Endfield-Optimization-for-Chen.git

# Install dependencies
npm install

# Run in development mode
npm run dev
```
